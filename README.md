# Mini-Trello Backend

NestJS backend for the Mini-Trello project. Provides REST API and WebSocket support for managing boards, columns, and tasks.

## Getting Started

### 1. Clone the repository

```bash
  git clone https://github.com/M1r0H/mini_trello_server.git
  cd mini_trello_server
```

### 2. Copy the environment variables

```bash
    cp .env.example .env
```

### 3. Run with Docker

```bash
  docker-compose up --build
```
